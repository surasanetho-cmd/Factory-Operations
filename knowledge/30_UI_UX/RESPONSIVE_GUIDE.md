# Responsive Guide

**Product:** Smart-Factory Manufacturing Platform  
**Companion:** [UI_STANDARD.md](UI_STANDARD.md) · [DESIGN_SYSTEM.md](DESIGN_SYSTEM.md)

---

## Breakpoints

| Name | Width | Shell behavior |
|------|-------|----------------|
| Mobile | &lt; 768px | Stack sidebar above content; list-first planning |
| Tablet | 768–1199px | Collapsible sidebar; touch-friendly drag handles |
| Desktop | ≥ 1200px | Persistent sidebar; full calendar + capacity boards |

---

## Planning board

| Viewport | Guidance |
|----------|----------|
| Desktop | Full line × day grid; drag-drop enabled |
| Tablet | Horizontal scroll on calendar; larger drop targets |
| Mobile | Prefer detail list + edit sheet; drag optional |

---

## Forms & tables

1. Tables scroll horizontally inside card frames — no page-wide overflow.  
2. Primary actions stay visible (sticky top bar on small screens).  
3. Touch targets ≥ 44px where possible.
